GLOBAL ATTRIBUTES :- Global attributes are attributes common to all elements;they can be used on all elements,though they may have no effect on some elements

Five Global Attributes
Style :- Specifies an inline CSS style for an element

title :-Specifies extra informatio about an element

Class :-Specifies one or more classname for an element

id :-Specifies a unique id for an element

dir :-Specifies the text direction for the content in an element
